Schülerdaten importieren Beachten Sie: Ein Import ist nur über eine csv-Datei möglicht.
Die Datei muss die Daten in folgender Reihenfolge beinhalten:
Name; Vorname; Jahrgangsstufe; Straße; PLZ; Ort; Geburtsdatum
Ein neuer Datensatz muss durch eine neue Zeile , ein neues Element muss mit ";" in der Datei gekennzeichnet werden.
Wählen Sie die gewünschte csv-Datei für den Datenimport aus.
